# 1️⃣ Project Idea (Best for Showcase)

“Smart Task Manager” 📱  
A productivity app with cloud sync, notifications, analytics, and offline support.

## Features

- Task management (CRUD)
- Categories & tags
- Offline-first database
- Cloud sync
- Push notifications
- Image attachments
- Settings with Remote Config
- Analytics dashboard

This allows you to demonstrate many Android skills in one project.
